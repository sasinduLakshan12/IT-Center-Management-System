### 1. Git සහ GitHub පාවිච්චි කරන ආකාරය (හොඳම ක්‍රමය)

   ```

   npm install
   ```

### පියවර 2: Backend (Node.js API) එක Deploy කිරීම

#### ක්‍රමය A: Backend එක Vercel එකේම Deploy කිරීම
Vercel එකේ backend එක දාන්න නම්, backend folder එක ඇතුළේ **`vercel.json`** කියලා configuration file එකක් හදන්න ඕනේ.

1. **`vercel.json` එකක් සෑදීම:** backend root folder එක ඇතුළේ මේ කේතය සහිතව `vercel.json` ලිපියක් සෑදිය යුතුය:
   ```json
   {
     "version": 2,
     "builds": [
       { "src": "server.js", "use": "@vercel/node" }
     ],
     "routes": [
       { "src": "/(.*)", "dest": "server.js" }
     ]
   }
   ```
2. **කේතය GitHub එකට Push කිරීම:** ප්‍රොජෙක්ට් එක GitHub එකට දාන්න.
3. **Vercel එකට සම්බන්ධ කිරීම:** Vercel Dashboard එකෙන් `New Project` ගොස් backend repository එක Import කරන්න.
4. **Environment Variables ඇතුළත් කිරීම:** Vercel එකේ Settings වල **Environment Variables** යටතේ backend එකේ `.env` ලිපියේ තිබුණු දේවල් ඇතුළත් කරන්න (විශේෂයෙන්ම `MONGO_URI`, `JWT_SECRET` ආදිය).
5. **Deploy:** Vercel විසින් ඔයාට Backend එක සඳහා Live URL එකක් ලබා දේවි (උදා: `https://my-backend.vercel.app`).

> [!TIP]

Frontend එක Vercel එකට දැමීම ඉතාමත් පහසුයි.

   * `Deploy` බොත්තම ඔබන්න. විනාඩියකින් පමණ ඔයාගේ වෙබ් අඩවිය Live වේවි! Vercel එකෙන් ඔයාට ලස්සන Live URL එකක් ලබා දේවි (උදා: `https://my-it-center.vercel.app`).
